# libdsa

C data structures and algorithms library, inspired on C++ STL.

## Features

- Opaque data types
- PIMPL design pattern
- C generic data structures leveraging void pointers
- 100% code coverage
- Memory-leak detection with Valgrind
- Google's C/C++ style

## Install

```bash
make release
make install
```
